# Application Instructions

## Pre-requisites
- Node.js (v20.14.0)
- Java 17
- Spring Boot 3.X
- Maven 3.X
- MySQL 8

## Steps to Run the Applications

### 1. Run the Front End Application

**Steps:**

1. Open the terminal.
2. Run the following command to start the application:

   ```bash
   npm run dev


### 2. Run the Back End Application

**Steps:**

1. Open the terminal.
2. Run the following command to start the application:

   ```bash
    mvn spring-boot:run
